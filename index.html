技能树（已点）

HTML5

语义性，熟练手写符合w3c标准的HTML页面。

CSS3

熟练使用CSS对网页进行（自适应）布局，了解不同浏览器的兼容性。

Bootstrap

很强大的一个前端框架，自适应布局，各种组件等。这是我学习的第一个前端UI库

jQuery ui

工作过程中学习的另一个前端ui框架，个人觉得相比bootstrap， jquery ui更加注重交互性，并且也能集成bootstrap样式，然而就是太重了。

Javascript

一直在追寻如何优雅的书写Javascript， 目前在研究ES6，这部分是我的一个学习重点。

jQuery

很强大的一个JS库，这是我学习的第一个javascript库，现在基本上已经熟练掌握。

node.js

之前公司的一个项目让我尝试使用了nodejs，一发不可的爱上了，新的项目暂时没有用到nodejs开发，目前我是拿来搭建前端自动化开发环境的。

Websocket

WebSocket protocol 是HTML5一种新的协议。它实现了浏览器与服务器全双工通信(full-duplex)，有幸在之前的项目中使用它进行实时数据收发。

Gulp

前端领域的构建工具（任务管理工具），通过简单的学习，已经能够熟练创建基本任务，配置相应的gulp配置文件。因为公司并不使用，所以很遗憾只是到会用。

Webpack

webpack是近期最火的一款模块加载器兼打包工具，它能把各种资源，都作为模块来使用和处理，配合Gulp使用事半功倍。

Sass

它提供了很多超棒的特性，已改进并增强了CSS的能力，然而需要编译是一大痛！所以需要学习gulp、webpack来改进并提高效率。

Photoshop

日常p个照片、切个图还行， 做设计的话。。我不是这个料。

Underscore.js

在使用nodejs开发后台的过程中， 学习使用了underscore，它让写js变得优美，让复杂的逻辑变得清晰。

Express

一个nodejs框架， 是我学习接触的第一个nodejs框架。

BaiduMap

一个前端的地图库， 公司项目中有许多地图业务， 我基本上对百度地图的API文档及使用都已掌握。

另一个开源的前端地图库，项目过程中有需求建立一套离线地图应用，所以抛弃了百度地图转向这个库。

Mock.js

一个开发工具，拦截ajax请求并返回模拟数据，方便前后端分离开发。

NPM

Bower

一个类似npm的存在， 只是它更专注于解决前端各种库的依赖管理。
